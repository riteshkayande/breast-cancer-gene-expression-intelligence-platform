📌 Project Overview

This project focuses on applying machine learning and computational genomics techniques to breast cancer gene expression datasets for disease intelligence and biomarker analysis.

The platform provides an interactive workflow for:

- Gene expression preprocessing
- Machine learning-based prediction
- Biomarker ranking
- PCA dimensionality reduction
- Multi-model evaluation
- Explainable AI-oriented analysis
- Interactive deployment through Streamlit

The system is designed to simulate a modern bioinformatics intelligence pipeline used in healthcare AI and computational biology environments.
