# AI-Driven Breast Cancer Gene Expression Intelligence Platform

## Overview
This project focuses on breast cancer disease prediction using gene expression data and machine learning techniques.

The platform combines:
- Bioinformatics
- Machine Learning
- Explainable AI
- Computational Genomics
- Data Visualization

## Features
- Gene expression preprocessing
- Feature selection
- PCA analysis
- Machine learning model comparison
- Explainable AI (SHAP)
- Streamlit dashboard
- API integration
- Docker deployment

## Tech Stack
- Python
- Scikit-learn
- XGBoost
- Pandas
- Streamlit
- SHAP
- Docker

## Models Used
- Logistic Regression
- Random Forest
- XGBoost
- SVM
